elements #

fuz_css applies default styles to semantic HTML elements in its üóé reset stylesheet. The styles use variables and include appropriate spacing, so plain HTML gets user-friendly styling and theme integration automatically. The defaults are low specificity using :where so they're easy to override, and you can opt out by adding .unstyled to an element.
